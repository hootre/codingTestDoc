# 조합 (Combinations)

> 조합은 n개 중에 r개를 뽑는 경우의 수를 구할 때 **순서를 고려하지 않는 개념**이다

## 예제

```javascript
Input: [1, 2, 3, 4];
Output: [
  [1, 2, 3],
  [1, 2, 4],
  [1, 3, 4],
  [2, 3, 4],
];
```

4C3은 4개중에 3개씩 선택할 때 나올 수 있는 모든 조합을 구한다는 말이다.
위 처럼 조합은 순서는 상관이없다. 즉 **[1, 2, 3] = [3, 2, 1]** 이렇게 순서가 바뀌어도
같은 구성이기 때문에 하나의 조합으로 취급한다.

구현

- 재귀
- 2중 for문
